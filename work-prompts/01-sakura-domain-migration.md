# 01: さくらインターネット 会員間ドメイン移行 (gck63819 → idd53821)

## ゴール
`quard-web.jp` ドメインを古い休眠アカウント **gck63819** から、新しい主運用アカウント **idd53821** へ移行する。

## 背景
- 2026-05-08 に誤って古いアカウント gck63819 で `quard-web.jp` を取得
- idd53821 にはレンタルサーバ Standard (`www2211.sakura.ne.jp`) と VPS (`133.242.136.126`) があり、ここで自社HPを公開したい
- 同一名義人 (高木 順) の所有なので会員間移行は可能なはず

## 前提
- gck63819 と idd53821 の両方の ID/パスワード把握済み
- 2要素認証コード受信可能
- 移行は承諾メール待ちで 1〜2日かかる場合あり

## ハンズオン手順

### Step 1: gck63819 でドメイン詳細画面を開く
1. https://secure.sakura.ad.jp/menu/ にアクセス
2. **gck63819** の ID/パスでログイン (※ idd53821 と別ブラウザ or プライベートウィンドウ推奨)
3. 左メニュー → **「契約情報」→「契約中のドメイン一覧」**
4. リストから **`quard-web.jp`** をクリック
5. 詳細画面の **操作リンクをすべて列挙** (例: 「ドメイン譲渡 / WHOIS情報変更 / 解約」等)

### Step 2: 移行/譲渡メニューを探す
詳細画面に以下のキーワードを含むメニューがあるか確認:
- 「ドメイン譲渡」「契約者変更」
- 「会員間移管」「サービス利用権譲渡」
- 「ドメイン移管」(さくら社内)

見つかった場合 → Step 3 へ
見つからない場合 → Step 5 (サポート問合せ) へ

### Step 3: 申請フォーム入力
1. 該当メニューをクリック
2. 譲渡先会員ID: **`idd53821`**
3. 譲渡先メールアドレス: idd53821 登録時のメール (Captain 確認)
4. 申請送信

### Step 4: idd53821 で承諾
1. idd53821 登録メールに「ドメイン譲渡承諾のお願い」等の通知が届く
2. メール本文のリンクをクリック → idd53821 でログイン → 承諾
3. 数時間〜1日で移行完了

### Step 5: メニューが見つからない場合 (サポート問合せ)
- 電話: 0120-775-664 (平日10-18時)
- 問合せフォーム: https://help.sakura.ad.jp/contact/
- 問合せ文面 (コピペ用):

```
お世話になっております。高木 順 と申します。

下記2つの会員IDは同一名義人(私)ですが、
誤って古い会員IDの方でドメインを取得してしまいました。
新しい運用用アカウントへドメインを譲渡したく、手順をご案内ください。

【譲渡元】会員ID: gck63819
【譲渡先】会員ID: idd53821
【対象ドメイン】quard-web.jp

どうぞよろしくお願いいたします。
```

## 成功判定
- idd53821 会員メニュー → 契約中のドメイン一覧に `quard-web.jp` が表示される
- gck63819 にはドメイン契約が無くなる (退会判断は後日)
